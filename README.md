# Post-Quantum Cryptography Simulation (Lattice-Based)

## Description
This project demonstrates a simple lattice-based encryption scheme inspired by Learning With Errors (LWE), a popular post-quantum cryptographic approach.

## Features
- Key generation (public & private)
- Encryption of binary message
- Decryption process
- Noise-based security (quantum resistant concept)

## Concept Used
- Lattice-based cryptography
- Learning With Errors (LWE)

## Technology
- Python
- NumPy

## How to Run
1. Open terminal in project folder
2. Run:
   python main.py

## Files
- main.py → Implementation
- README.md → Documentation
- report.pdf → Detailed explanation

## Output
- Original message
- Decrypted message
